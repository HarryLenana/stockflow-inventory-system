# StockFlow Inventory Management System

## Overview

StockFlow is a modern inventory management system built using Flask and SQLite.

The application helps businesses manage products, suppliers, purchases, sales and inventory analytics through a professional dashboard.

## Features

* User Authentication
* Product Management
* Supplier Management
* Purchase Tracking
* Sales Tracking
* Dashboard Analytics
* Revenue Monitoring
* Inventory Value Tracking
* CSV Export
* Reports Module
* Responsive Interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML5
* CSS3
* JavaScript
* Chart.js

## Installation

```bash
pip install -r requirements.txt
python app.py
```
